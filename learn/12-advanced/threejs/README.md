# Three.js — Audio Visualization

## Overview

Three.js provides GPU-accelerated 3D rendering in the browser. DJ.ai plans to use it for real-time audio visualization driven by frequency analysis data.

## Learning Path

| File | Topic |
|------|-------|
| [audio-visualization.md](./audio-visualization.md) | WebGL scene + Web Audio API integration |

## Key Concepts

- **Scene** — Container for 3D objects
- **Camera** — Viewer perspective
- **Renderer** — Draws to canvas via WebGL
- **AnalyserNode** — Web Audio API frequency data source
